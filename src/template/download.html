百度网盘资源破解下载说明

当你在百度网盘里面找到了一个想要看的电影或资料，但是它是个压缩包，那你就会去下载，然而当你发现百度网盘客户端的下载速度只有几十K的时候你是不是会很崩溃？ 这里提供一种方法，让你不需要购买会员也能高速下载资源。

1.把要下载的文件保存到自己的百度网盘中

2.下载客户端

客户端下载

客户端下载

Windows

程序应在 命令提示符 (Command Prompt) 或 PowerShell 中运行, 在 mintty (例如: GitBash) 可能会有显示问题.

Linux / macOS

程序应在 终端 (Terminal) 运行.

Android / iOS

Android / iOS 移动设备操作比较麻烦, 不建议在移动设备上使用本程序.

3.运行客户端

登录百度帐号

常规登录百度帐号

支持在线验证绑定的手机号或邮箱,

BaiduPCS-Go login

使用百度 BDUSS 来登录百度帐号
关于 获取百度 BDUSS

BaiduPCS-Go login -bduss=$BDUSS

退出百度帐号

BaiduPCS-Go logout

切换工作目录

BaiduPCS-Go cd 目录

下载文件/目录

默认下载方式

BaiduPCS-Go download 网盘文件或目录的路径

直链下载方式

BaiduPCS-Go download --locate 网盘文件或目录的路径

直链负载均衡下载方式（速度最快，推荐使用）

BaiduPCS-Go download --locate_pan 网盘文件或目录的路径
